Justify Content Exercise

In this exercise, you'll practice the justify-content property. This property controls how flex items are distributed along the main axis. The container below has four items. Currently, justify-content is set to flex-start.

Tasks:

  1. Open style.css and change justify-content to flex-end. Observe how items move to the end of the container.
  2. Change it to center. See items centered in the container.
  3. Change it to space-between. Notice equal space between items.
  4. Change it to space-around. See space around each item.
  5. Change it to space-evenly. Observe even spacing including edges.
  6. Experiment with different values and see the effects.
