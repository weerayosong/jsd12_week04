Have students add Flexbox code in style.css following these instructions:

  1. Open style.css.
  2. Enable display: flex on .navbar.
  3. Use justify-content to push the menu items apart (logo left, menu right).
  4. Use align-items to vertically center everything.
  5. Use .menu span with margin-left to add space between menu links.
