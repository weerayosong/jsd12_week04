Flex Direction Exercise

In this exercise, you'll practice the flex-direction property. The container below has four items (A, B, C, D). Currently, flex-direction is set to row.

Tasks:

  1. Open style.css and change flex-direction to column. Observe how the items arrange vertically.
  2. Change it to row-reverse. Notice the order of the items.
  3. Change it to column-reverse. See the vertical reverse order.
  4. Experiment with different values and see the effects.
