Flex Order Exercise

In this exercise, you'll practice the order property. This property controls the order of flex items without changing the HTML. The container below has four items labeled A, B, C, D.

Tasks:

  1. Open style.css and set order: 1 for item B. Observe how it moves to the end.
  2. Set order: -1 for item C. See how it moves to the beginning.
  3. Set order: 2 for item D. Notice the new order: C, A, B, D.
  4. Try different order values and see how items rearrange.
  5. Experiment with negative and positive orders.
