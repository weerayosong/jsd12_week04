Flex Item Sizing Exercise

In this exercise, you'll practice the flex-grow, flex-shrink, and flex-basis properties. These control how flex items grow, shrink, and their initial size. The container below has four items with different initial widths.

Tasks:

  1. Open style.css and set flex-grow to 1 for all items. Observe how they grow to fill available space equally.
  2. Set flex-grow to 2 for the second item. See how it grows twice as much as others.
  3. Set flex-shrink to 0 for the third item. Resize the browser to see it doesn't shrink below its basis.
  4. Set flex-basis to 200px for the fourth item. Notice its starting size before growing.
  5. Experiment with different combinations of these properties.
