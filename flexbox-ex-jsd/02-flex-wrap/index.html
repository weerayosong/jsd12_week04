Flex Wrap Exercise

In this exercise, you'll practice the flex-wrap property. The container below has eight items. Currently, flex-wrap is set to nowrap, so all items stay in one row even if they overflow.

Tasks:

  1. Open style.css and change flex-wrap to wrap. Observe how the items wrap to new lines when there's not enough space.
  2. Change it to wrap-reverse. Notice how the wrapped lines appear in reverse order.
  3. Try resizing your browser window to see how wrapping responds to different container widths.
  4. Experiment with different values and see the effects.
