Align Self Exercise

In this exercise, you'll practice the align-self property. This property allows individual flex items to override the container's align-items value. The container below has four items of different heights.

Tasks:

  1. Open style.css and set align-self: flex-start for item B. Observe how it aligns to the top.
  2. Set align-self: flex-end for item C. See how it aligns to the bottom.
  3. Set align-self: center for item D. Notice it centers vertically.
  4. Set align-self: stretch for item A. See how it stretches to fill the container height.
  5. Experiment with different values for each item.
